Tools
=====
    
# Requierements
```
export GALAXY_ROOT=/path/to/galaxy
git clone https://github.com/galaxyproject/galaxy.git $GALAXY_ROOT
cd $GALAXY_ROOT
git checkout v17.05
```
# Installation
```
git clone https://github.com/destairdenbi/tools.git
cd tools
./setup.sh
```

# RUN
```
$GALAX_ROOT/run.sh
```
